# My NEOVIM configuration

simple neovim setup with manual plugin handling

## How to setup?

Create on linux

  mkdir -p ~/.config/nvim
  mkdir -p ~/.config/nvim/pack/plugins

on windows

  mkdir -p $env:LOCALAPPDATA/nvim
  mkdir -p $env:LOCALAPPDATA/nvim/pack/plugins
